sparkline.js

Library for drawing Sparklines, using SVG.

Objects

Methods

  • OAT.Sparkline::attachData(dataArray)

OAT.Sparkline(div, optObj)

Creates a Sparkline inside div, optObj describes its properties. Its values are the same as in Line Chart.

OAT.Sparkline::attachData(dataArray)

Adds data to sparkline.

OAT.Sparkline::draw()

Draws the Sparkline.

var s = new OAT.Sparkline("myDiv",{});
s.attachData([1,2,3,-5,2,3.7,5,4]);
s.draw();
