OAT.Connection(type, optionsObject)

Creates a Connection. type is one of mentioned type constants. Available options depend on connection type:

  • TYPE_XMLA
    • endpoint
    • dsn
    • user
    • password
  • TYPE_REST
    • url
  • TYPE_WSDL
    • url
