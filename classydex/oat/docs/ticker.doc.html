ticker.js

Ticker - slowly appearing (and disappearing) text.

Objects

  • OAT.Ticker(element, text, optObj)

Methods

  • OAT.Ticker.start()
  • OAT.Ticker.stop()

Constants

  • OAT.TickerData.ADD_START - add characters to the beginning
  • OAT.TickerData.ADD_END - add characters to end
  • OAT.TickerData.CLEAR_ALL - remove all characters at once
  • OAT.TickerData.CLEAR_START - remove characters from beginning
  • OAT.TickerData.CLEAR_END - remove characters from end
  • OAT.TickerData.TIMING_PERCHAR - delay is specified per character
  • OAT.TickerData.TIMING_GLOBAL - delay is specified per string
  • OAT.TickerData.LOOP_NONE - don't loop
  • OAT.TickerData.LOOP_BACK - draw text, hide it and stop
  • OAT.TickerData.LOOP_FULL - loop

OAT.Ticker(element, text, optObj)

Creates Ticker. It will show text in element. Argument optObj is an object containing the following:

  • loop - a LOOP_ constant
  • add - an ADD_ constant
  • clear - a CLEAR_ constant
  • timing - a TIMING_ constant
  • delay - delay in msec
  • pause - pause (in msec) when all text is drawn

var t = new OAT.Ticker(div, "Hello world!", {delay:30,timing:TickerData.TIMING_PERCHAR});
t.start();

OAT.Ticker::start()

Starts ticking.

OAT.Ticker::stop()

Stops ticking.
